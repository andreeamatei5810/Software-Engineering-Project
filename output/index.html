Operations

  • PUB sensorId/crop

    Accepts the following message:

    Publish cropPublishCrop

    Each time a new crop is added, a message with the contents of the crop is published.

    object
    id
    string

    Crop id

    sensorId
    string

    The id of the sensor to which the crop is associated

    timeStamp
    string
    format: date-time

    The date the crop information was taken over

    name
    string

    Crop name

    height
    number

    The height of the plants in the crop

    leafColor
    string

    Crop leaf color

    leafTemperature
    number

    Crop leaf temperature

    Additional properties are allowed.

    Examples

    {
  "id": "string",
  "sensorId": "string",
  "timeStamp": "2019-08-24T14:15:22Z",
  "name": "string",
  "height": 0,
  "leafColor": "string",
  "leafTemperature": 0
}

  • PUB sensorId/weather

    Accepts the following message:

    Publish weatherPublishWeather

    Each time a new weather is added, a message with the contents of the weather is published.

    object
    id
    string

    Weather id

    sensorId
    string

    The id of the sensor to which the weather is associated

    timeStamp
    string
    format: date-time

  • PUB irrigation

    Accepts the following message:

    Publish irrigationPublishIrrigation

    Each time the irrigation system is turned off / on by the device a message is published with the details of the irrigation system.

    object
    id
    string

    Weather id

    waterAmount
    integer

    The amount of water needed for the soil to be given by the irrigation system

    timeStamp
    string
    format: date-time

    Date on which the irrigation system is switched off / on

    power
    boolean

    The irrigation system is turned off or on

    object

    The soil associated with the irrigation system

    Additional properties are allowed.

    Additional properties are allowed.

    Examples

    {
  "id": "string",
  "waterAmount": 0,
  "timeStamp": "2019-08-24T14:15:22Z",
  "power": true,
  "soil": {}
}
